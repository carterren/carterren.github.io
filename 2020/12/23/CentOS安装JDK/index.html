2020-12-23 Comments

基于CentOS安装JDK

1.卸载系统自带的JDK

  • 1.2先查看一下自带的jdk java -version

  • 1.2查看java相关文件

  • 1.3 删除相关文件

    ​ yum -y remove java-1.8.0-openjdk-headless.x86_64

​ yum -y remove java-1.8.0-openjdk-1.8.0.181-7.b13.el7.x86_64

2.安装jdk和环境变量配置

  • 2.1切换到 /usr/local 新建java文件夹（mkdir java）

  • 2.2rz命令上传文件到新建的java文件夹

  • 2.3 tar -zxvf jdk-15.0.1_linux-x64_bin.tar.gz 解压文件夹 解压之后可以删除之前的压缩包
  • 2.4随后配置环境变量 vim /etc/profile 按 i 进入编辑模式在最后键入,esc退出编辑模式 shift+zz 保存并退出
  • 2.5然后键入命令source /etc/profile 使配置文件生效

3.测试安装情况 java -version 出现版本号就是成功了
